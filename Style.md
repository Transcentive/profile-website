# Style.md

House style rules for this site's copy. Read this before writing or editing any prose.

## Punctuation

- **No em-dashes (—). Ever.** The em-dash is verboten. It is also a tell of AI-generated text.
  Recast the sentence instead:
  - Use a **comma** for a light aside or pause.
  - Use a **colon** when the second clause explains or expands the first.
  - Use **parentheses** for a true aside.
  - Use a **period** and split into two sentences when the dash was joining independent clauses.
  - Use a spaced **en-dash ( – )** only for genuine ranges (e.g. 2019 – 2024), never as a sentence break.
- Do not swap an em-dash for a spaced hyphen or a bare en-dash as a substitute. Rewrite the sentence.

## Applies to

All copy in `site/*.html` and any CSS-generated content (e.g. `::before { content: ... }`).
